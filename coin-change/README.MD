# 322. Coin Change

**Сложность:** Medium  
**Источник:** [LeetCode](https://leetcode.com/problems/coin-change/)  
**Темы:** Array, Dynamic Programming, BFS

## ⚠️ Примечание

Данная реализация использует **BFS-подход** и **не является оптимальным решением**. 

Оптимальное решение этой задачи использует **динамическое программирование** со сложностью O(amount × coins.length). BFS-решение может быть медленнее на больших входных данных из-за посещения большего количества состояний.

## Описание

Даны монеты различных номиналов и общая сумма. Необходимо найти **минимальное количество монет**, которое нужно для набора этой суммы.

Если набрать сумму невозможно, вернуть `-1`.

**Предполагается**, что монет каждого номинала неограниченно много.

## Алгоритм (BFS)

Используем поиск в ширину для нахождения кратчайшего пути до нужной суммы:

1. **Инициализация:** Добавляем все номиналы монет в очередь
2. **BFS по уровням:** Каждый уровень — это использование ещё одной монеты
3. **Посещённые состояния:** Используем HashSet для избежания повторной обработки одинаковых сумм
4. **Первый успех:** Когда достигаем нужной суммы — возвращаем текущий шаг

## Визуализация

**Пример 1:**
```
Вход:  coins = [1, 2, 5], amount = 11
       ↓
Выход: 3 (11 = 5 + 5 + 1)

Шаги BFS:
Уровень 1: [1, 2, 5]
Уровень 2: [1+1=2, 1+2=3, 1+5=6, 2+2=4, 2+5=7, 5+5=10]
Уровень 3: [..., 5+5+1=11] ✓ → ответ: 3
```

**Пример 2:**
```
Вход:  coins = [2], amount = 3
       ↓
Выход: -1 (невозможно набрать)

Шаги BFS:
Уровень 1: [2]
Уровень 2: [2+2=4] > 3, пропускаем
Очередь пуста → ответ: -1
```

**Пример 3:**
```
Вход:  coins = [1], amount = 0
       ↓
Выход: 0 (сумма 0 не требует монет)

Шаги:
amount == 0 → сразу возвращаем 0
```

**Пример 4:**
```
Вход:  coins = [1, 3, 4], amount = 6
       ↓
Выход: 2 (6 = 3 + 3)

Шаги BFS:
Уровень 1: [1, 3, 4]
Уровень 2: [1+1=2, 1+3=4, 1+4=5, 3+3=6] ✓ → ответ: 2
```

## Примеры

| Монеты | Сумма | Выход | Пояснение |
|--------|-------|-------|-----------|
| `[1,2,5]` | 11 | 3 | `5 + 5 + 1` |
| `[2]` | 3 | -1 | Невозможно |
| `[1]` | 0 | 0 | Пустая сумма |
| `[1]` | 1 | 1 | `1` |
| `[1,3,4]` | 6 | 2 | `3 + 3` |
| `[1,2,5]` | 100 | 20 | `20 × 5` |
