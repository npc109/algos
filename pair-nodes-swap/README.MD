# 24. Swap Nodes in Pairs

**Сложность:** Medium  
**Источник:** [LeetCode](https://leetcode.com/problems/swap-nodes-in-pairs/)  
**Темы:** Linked List, Recursion

## Описание

Дан связный список. Необходимо поменять местами каждые два соседних узла и вернуть голову изменённого списка.

**Важно:** Менять можно только сами узлы, значения внутри узлов изменять нельзя.

## Алгоритм

Рекурсивный подход:
1. Если есть хотя бы два узла — меняем их местами
2. Первый узел становится вторым, второй — первым
3. Рекурсивно применяем алгоритм к остальной части списка

## Визуализация

**Пример 1:**
```
Вход:  1 → 2 → 3 → 4
       ↓   ↓   ↓   ↓
       ↓   ↓   ↓   ↓
Выход: 2 → 1 → 4 → 3
```

**Пример 2:**
```
Вход:  1 → 2 → 3
       ↓   ↓   ↓
       ↓   ↓   ↓
Выход: 2 → 1 → 3
```

**Пример 3 (пустой список):**
```
Вход:  []
Выход: []
```

**Пример 4 (один элемент):**
```
Вход:  1
Выход: 1
```

## Примеры

| Вход | Выход |
|------|-------|
| `[1,2,3,4]` | `[2,1,4,3]` |
| `[]` | `[]` |
| `[1]` | `[1]` |
| `[1,2,3]` | `[2,1,3]` |
