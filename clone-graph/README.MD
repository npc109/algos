# 133. Clone Graph

**Сложность:** Medium  
**Источник:** [LeetCode](https://leetcode.com/problems/clone-graph/)  
**Темы:** Hash Table, Graph, DFS, BFS

## Описание

Дан узел неориентированного связного графа. Необходимо создать **глубокую копию** (клон) этого графа.

Каждый узел содержит значение (`val`) и список соседей (`neighbors`).

**Важно:**
- Граф может содержать циклы
- Клон должен быть независимой копией (никаких ссылок на оригинальные узлы)
- Все значения узлов уникальны

## Алгоритм (DFS с HashMap)

Используем рекурсивный обход в глубину с мемоизацией:

1. **HashMap `visited`** — хранит соответствие `оригинальное значение → клонированный узел`
2. **Базовый случай:** Если узел уже посещён — возвращаем его клон из HashMap
3. **Рекурсивный шаг:**
   - Создаём новый узел с тем же значением
   - Добавляем в HashMap
   - Рекурсивно клонируем всех соседей

## Визуализация

**Пример 1 (граф из 4 узлов):**
```
Вход:  adj = [[2,4],[1,3],[2,4],[1,3]]
       ↓
Граф:
       1
     /   \
    2 ——  4
     \   /
       3

Выход: Клонированный граф с той же структурой

Шаги DFS:
→ cloneGraph(1): создаём узел 1, visited = {1: 1'}
  → cloneGraph(2): создаём узел 2, visited = {1: 1', 2: 2'}
    → cloneGraph(1): уже в visited → возвращаем 1'
    → cloneGraph(3): создаём узел 3, visited = {1: 1', 2: 2', 3: 3'}
      → cloneGraph(2): уже в visited → возвращаем 2'
      → cloneGraph(4): создаём узел 4, visited = {..., 4: 4'}
        → cloneGraph(1): уже в visited → возвращаем 1'
        → cloneGraph(3): уже в visited → возвращаем 3'
  
Результат: 1'→[2',4'], 2'→[1',3'], 3'→[2',4'], 4'→[1',3']
```

**Пример 2 (один узел):**
```
Вход:  adj = [[]]
       ↓
Граф:  1 (без соседей)

Выход: 1 (клон без соседей)
```

**Пример 3 (два узла):**
```
Вход:  adj = [[2],[1]]
       ↓
Граф:  1 —— 2

Выход: 1' —— 2'
```

**Пример 4 (пустой граф):**
```
Вход:  null
       ↓
Выход: null
```

## Формат входных данных

Граф задаётся через список смежности `adj`, где `adj[i]` содержит список соседей узла с значением `i+1`.

| adj | Граф |
|-----|------|
| `[[2,4],[1,3],[2,4],[1,3]]` | 1-2, 1-4, 2-3, 3-4 (квадрат) |
| `[[]]` | 1 (изолированный узел) |
| `[[2],[1]]` | 1-2 (два узла) |
| `[[2,3],[1,3],[1,3]]` | 1-2, 1-3, 2-3 (треугольник) |

## Примеры

| Вход (adj) | Выход (структура) |
|------------|-------------------|
| `[[2,4],[1,3],[2,4],[1,3]]` | Клон квадрата 1-2-3-4 |
| `[[]]` | Клон изолированного узла |
| `[[2],[1]]` | Клон двух соединённых узлов |
| `[[2,3],[1,3],[1,3]]` | Клон треугольника |
| `[]` | `null` (пустой граф) |
