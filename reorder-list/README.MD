# 143. Reorder List

**Сложность:** Medium  
**Источник:** [LeetCode](https://leetcode.com/problems/reorder-list/)  
**Темы:** Linked List, Two Pointers, Fast & Slow Pointers

## Описание

Дан односвязный список. Необходимо перестроить его элементы по следующему правилу:

```
L0 → Ln → L1 → Ln-1 → L2 → Ln-2 → ...
```

**Важно:**
- Изменения должны выполняться на месте (in-place)
- Нельзя просто менять значения узлов — нужно менять сами связи

## Алгоритм

Задача решается в три этапа:

1. **Найти середину списка:** Используем быстрый и медленный указатели (fast & slow)
2. **Развернуть вторую половину:** Разворачиваем часть списка после середины
3. **Слить две половины:** Чередуем узлы из первой и второй половины

## Визуализация

**Пример 1:**
```
Вход:  1 → 2 → 3 → 4 → 5
       ↓
Выход: 1 → 5 → 2 → 4 → 3

Шаг 1: Найти середину
  slow=1, fast=2
  slow=2, fast=4
  slow=3 (середина)

Шаг 2: Развернуть вторую половину (после 3)
  Первая половина:  1 → 2 → 3 → null
  Вторая половина:  5 → 4 → null  (развёрнуто 4→5)

Шаг 3: Слить две половины
  1 → 5 → 2 → 4 → 3
```

**Пример 2:**
```
Вход:  1 → 2 → 3 → 4
       ↓
Выход: 1 → 4 → 2 → 3

Шаг 1: Найти середину
  slow=1, fast=2
  slow=2 (середина для чётной длины)

Шаг 2: Развернуть вторую половину (после 2)
  Первая половина:  1 → 2 → null
  Вторая половина:  4 → 3 → null  (развёрнуто 3→4)

Шаг 3: Слить две половины
  1 → 4 → 2 → 3
```

**Пример 3 (пошаговая трассировка):**
```
Вход:  1 → 2 → 3 → 4 → 5 → 6

Шаг 1: Найти середину
  Итерация | slow | fast
  ---------|------|------
    0      |  1   |  2
    1      |  2   |  4
    2      |  3   |  6
  Середина: 3

Шаг 2: Развернуть вторую половину
  До: 4 → 5 → 6 → null
  После: 6 → 5 → 4 → null

Шаг 3: Слить две половины
  Итерация | h (первая) | fast (вторая) | Результат
  ---------|------------|---------------|------------------
    0      |     1      |       6       | 1→6
    1      |     2      |       5       | 1→6→2→5
    2      |     3      |       4       | 1→6→2→5→3→4

Выход: 1 → 6 → 2 → 5 → 3 → 4
```

**Пример 4 (один элемент):**
```
Вход:  1 → null
       ↓
Выход: 1 → null

Шаги:
head.next==null → сразу возвращаем head
```

## Примеры

| Вход | Выход |
|------|-------|
| `[1,2,3,4,5]` | `[1,5,2,4,3]` |
| `[1,2,3,4]` | `[1,4,2,3]` |
| `[1,2,3,4,5,6]` | `[1,6,2,5,3,4]` |
| `[1]` | `[1]` |
| `[1,2]` | `[1,2]` |
