# 80. Remove Duplicates from Sorted Array II

**Сложность:** Medium  
**Источник:** [LeetCode](https://leetcode.com/problems/remove-duplicates-from-sorted-array-ii/)  
**Темы:** Array, Two Pointers

## Описание

Дан отсортированный массив целых чисел. Необходимо удалить дубликаты так, чтобы каждый элемент встречался **не более двух раз**, и вернуть новую длину массива.

**Важно:**
- Изменения должны выполняться на месте (in-place)
- Не использовать дополнительную память
- Первые `k` элементов массива должны содержать результат

## Алгоритм

Используем подход с двумя указателями:
1. **`w` (write)** — позиция для записи следующего элемента (начинается с 2, т.к. первые 2 элемента всегда остаются)
2. **`r` (read)** — текущий читаемый элемент (начинается с 2)
3. Сравниваем `nums[r]` с `nums[w-2]`:
   - Если они разные — элемент можно записать (он не нарушает условие)
   - Если одинаковые — пропускаем (это третий дубликат)

## Визуализация

**Пример 1:**
```
Вход:  [1, 1, 1, 2, 2, 3]
       ↓
Выход: k = 5, первые 5 элементов: [1, 1, 2, 2, 3, _]

Шаги:
[1, 1, 1, 2, 2, 3] → первые 2 элемента (1,1) остаются
→ r=2: nums[2]=1, nums[w-2]=nums[0]=1 → одинаково, пропускаем
→ r=3: nums[3]=2, nums[w-2]=nums[0]=1 → разные, записываем
→ r=4: nums[4]=2, nums[w-2]=nums[1]=1 → разные, записываем
→ r=5: nums[5]=3, nums[w-2]=nums[2]=2 → разные, записываем
→ результат: [1, 1, 2, 2, 3, _], k = 5
```

**Пример 2:**
```
Вход:  [0, 0, 1, 1, 1, 1, 2, 3, 3]
       ↓
Выход: k = 7, первые 7 элементов: [0, 0, 1, 1, 2, 3, 3, _, _]

Шаги:
[0, 0, 1, 1, 1, 1, 2, 3, 3] → первые 2 элемента (0,0) остаются
→ r=2: nums[2]=1 ≠ nums[0]=0 → записываем
→ r=3: nums[3]=1 ≠ nums[1]=0 → записываем
→ r=4: nums[4]=1 = nums[2]=1 → пропускаем (третий дубликат)
→ r=5: nums[5]=1 = nums[2]=1 → пропускаем
→ r=6: nums[6]=2 ≠ nums[2]=1 → записываем
→ r=7: nums[7]=3 ≠ nums[3]=1 → записываем
→ r=8: nums[8]=3 ≠ nums[4]=2 → записываем
→ результат: [0, 0, 1, 1, 2, 3, 3, _, _], k = 7
```

**Пример 3 (все элементы одинаковые):**
```
Вход:  [5, 5, 5, 5, 5]
       ↓
Выход: k = 2, первые 2 элемента: [5, 5, _, _, _, _]

Шаги:
[5, 5, 5, 5, 5] → первые 2 элемента остаются
→ r=2,3,4: все равны nums[w-2] → пропускаем
→ результат: [5, 5, _, _, _], k = 2
```

## Примеры

| Вход | Выход (k) | Первые k элементов |
|------|-----------|-------------------|
| `[1,1,1,2,2,3]` | 5 | `[1,1,2,2,3]` |
| `[0,0,1,1,1,1,2,3,3]` | 7 | `[0,0,1,1,2,3,3]` |
| `[5,5,5,5,5]` | 2 | `[5,5]` |
| `[1,2,3]` | 3 | `[1,2,3]` |
