# 31. Next Permutation

**Сложность:** Medium  
**Источник:** [LeetCode](https://leetcode.com/problems/next-permutation/)  
**Темы:** Array, Two Pointers

## Описание

Дан массив целых чисел. Необходимо найти следующую лексикографически большую перестановку этого массива.

Если следующая большая перестановка невозможна (массив отсортирован по убыванию), необходимо преобразовать массив в наименьшую перестановку (отсортировать по возрастанию).

**Важно:** Изменения должны выполняться на месте (in-place), без использования дополнительной памяти.

## Алгоритм

1. **Найти точку перелома:** Идём справа налево, ищем первый элемент, который меньше следующего (`nums[i-1] < nums[i]`)
2. **Если точки перелома нет:** Массив отсортирован по убыванию — разворачиваем весь массив
3. **Если точка перелома найдена:**
   - Находим справа элемент, который больше `nums[breakPoint-1]`
   - Меняем их местами
   - Разворачиваем часть массива после точки перелома

## Визуализация

**Пример 1:**
```
Вход:  [1, 2, 3]
       ↓
Выход: [1, 3, 2]

Шаги:
[1, 2, 3] → точка перелома на индексе 2 (3 > 2)
→ меняем 2 и 3
→ [1, 3, 2]
```

**Пример 2:**
```
Вход:  [3, 2, 1]
       ↓
Выход: [1, 2, 3]

Шаги:
[3, 2, 1] → нет точки перелома (убывающий порядок)
→ разворачиваем весь массив
→ [1, 2, 3]
```

**Пример 3:**
```
Вход:  [1, 1, 5]
       ↓
Выход: [1, 5, 1]

Шаги:
[1, 1, 5] → точка перелома на индексе 2 (5 > 1)
→ меняем 1 и 5
→ [1, 5, 1]
```

**Пример 4:**
```
Вход:  [1, 3, 2]
       ↓
Выход: [2, 1, 3]

Шаги:
[1, 3, 2] → точка перелома на индексе 1 (3 > 1)
→ находим элемент > 1 справа (это 2)
→ меняем 1 и 2
→ разворачиваем [3]
→ [2, 1, 3]
```

**Пример 5 (из теста 7):**
```
Вход:  [4, 2, 0, 2, 3, 2, 0]
       ↓
Выход: [4, 2, 0, 3, 0, 2, 2]

Шаги:
[4, 2, 0, 2, 3, 2, 0] → точка перелома на индексе 4 (3 > 2)
→ находим элемент > 2 справа (это 3)
→ меняем 2 и 3
→ разворачиваем [2, 2, 0] → [0, 2, 2]
→ [4, 2, 0, 3, 0, 2, 2]
```

## Примеры

| Вход | Выход |
|------|-------|
| `[1,2,3]` | `[1,3,2]` |

| `[1,3,2]` | `[2,1,3]` |
| `[1]` | `[1]` |
| `[2,3,1]` | `[3,1,2]` |
| `[4,2,0,2,3,2,0]` | `[4,2,0,3,0,2,2]` |
