# 206. Reverse Linked List

**Сложность:** Easy  
**Источник:** [LeetCode](https://leetcode.com/problems/reverse-list/)  
**Темы:** Linked List, Recursion, Iteration

## Описание

Дан односвязный список. Необходимо развернуть его в обратном направлении и вернуть новую голову списка.

**Важно:** Изменения должны выполняться на месте (in-place), без использования дополнительной памяти.

## Алгоритм (итеративный подход)

Используем три указателя для разворота списка за один проход **O(n)**:

1. **`node`** — предыдущий узел (изначально `null`)
2. **`head`** — текущий узел
3. **`tmp`** — следующий узел (сохраняем перед изменением ссылки)

На каждом шаге:
- Сохраняем `head.next` во `tmp`
- Разворачиваем ссылку: `head.next = node`
- Сдвигаем `node` на `head`
- Сдвигаем `head` на `tmp`

## Визуализация

**Пример 1:**
```
Вход:  1 → 2 → 3 → 4 → 5 → null
       ↓
Выход: 5 → 4 → 3 → 2 → 1 → null

Пошаговая трассировка:

Шаг 0:
  node=null, head=1→2→3→4→5

Шаг 1:
  tmp=2, head.next=null, node=1, head=2
  Список: null←1  2→3→4→5

Шаг 2:
  tmp=3, head.next=1, node=2, head=3
  Список: null←1←2  3→4→5

Шаг 3:
  tmp=4, head.next=2, node=3, head=4
  Список: null←1←2←3  4→5

Шаг 4:
  tmp=5, head.next=3, node=4, head=5
  Список: null←1←2←3←4  5

Шаг 5:
  tmp=null, head.next=4, node=5, head=null
  Список: null←1←2←3←4←5

Возвращаем: node=5→4→3→2→1→null
```

**Пример 2:**
```
Вход:  1 → 2 → null
       ↓
Выход: 2 → 1 → null

Шаги:
→ Шаг 1: tmp=2, head.next=null, node=1, head=2
→ Шаг 2: tmp=null, head.next=1, node=2, head=null
Возвращаем: 2→1→null
```

**Пример 3 (пустой список):**
```
Вход:  null
       ↓
Выход: null

Шаги:
head==null → сразу возвращаем null
```

**Пример 4 (один элемент):**
```
Вход:  1 → null
       ↓
Выход: 1 → null

Шаги:
→ Шаг 1: tmp=null, head.next=null, node=1, head=null
Возвращаем: 1→null
```

## Примеры

| Вход | Выход |
|------|-------|
| `[1,2,3,4,5]` | `[5,4,3,2,1]` |
| `[1,2]` | `[2,1]` |
| `[]` | `[]` |
| `[1]` | `[1]` |
